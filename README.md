# 어비 강사 온보딩 센터

(주)어비 강사가 계약 직후부터 첫 강의, 코치방 운영, 수강생 관리까지 필요한 업무를 순서대로 준비할 수 있는 웹 기반 운영 가이드입니다.

## 주요 기능

- 강사별 계약·등급·전문 분야·육성 진행률 대시보드
- 7단계 강사 육성 로드맵과 완료 기준
- 8단계 강의 기획안 작성·임시저장·검토 요청
- 강의 전·중·후 운영 체크리스트
- Zoom 리허설 및 카카오 코치방 개설 가이드
- 1·2·3단계 위기 대응 매뉴얼과 이슈 신고
- 별도 관리자 페이지(`/admin`)에서 신규 강사와 기본 온보딩 단계 등록
- 강사가 요청한 전자책·템플릿·강의자료를 파일 또는 링크로 개인 자료실에 전달
- ChatGPT 로그인 기반 사용자 식별과 D1 기반 영속 저장

## 로컬 실행

Node.js 22.13 이상이 필요합니다.

```bash
npm ci
npm run dev
```

`http://localhost:3000`에서 확인할 수 있습니다. 로컬에서는 데모 강사 계정이 자동으로 사용되며, 배포 환경에서는 인증된 사용자 정보가 적용됩니다.

## 검증

```bash
npm run build
node --test tests/rendered-html.test.mjs
```

## 데이터

강사 계정, 프로필, 진행 단계, 기획안, 위기 이슈와 자료 전달 이력은 Cloudflare D1에 저장됩니다. 업로드 파일은 `FILES` R2 버킷에 저장하며, 권한이 있는 관리자와 대상 강사만 서버 경유로 열 수 있습니다. 스키마와 배포용 마이그레이션은 `db/`와 `drizzle/`에 있습니다.
